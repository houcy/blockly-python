Health Java Library

From the CORGIS Dataset Project

By Austin Cory Bart (acbart@vt.edu)
Version 1, created 11/1/2015
Tags: health, diseases, infection

Overview

This library holds data about reports of diseases in America.

Downloads

Download all of the following files.

  1. health-doc-1.jar
  2. health.db

Usage

Begin by adding the above JAR file to your classpath. This library has 1 function you can use. 
import corgis.health.HealthLibrary;
import corgis.health.domain.*;
import java.util.ArrayList;

public class ExampleClass {
    public static void main(String[] args) {
        // Get access to the library
        HealthLibrary healthLibrary = new HealthLibrary();
        // Access data inside the library
        ArrayList<Report> list_of_report = healthLibrary.getAllReports(false);
    }
}
Additionally, some of the functions can optionally return Big Data using the last argument (setting it to true). If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import corgis.health.HealthLibrary;
import corgis.health.domain.*;
import java.util.ArrayList;

public class ExampleClass {
    public static void main(String[] args) {
        // Get access to the library
        HealthLibrary healthLibrary = new HealthLibrary();
        // Access data inside the library
        ArrayList<Report> list_of_report = healthLibrary.getAllReports(true);
    }
}

Documentation

We provide full JavaDoc for this library. Alternatively, you may refer to the below:

 healthLibrary.getAllReports(test)

Returns information about the status of disease for all locations, diseases, and years.

  • optional
    test: boolean

    Argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
