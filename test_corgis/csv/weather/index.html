Overview

Under the National Oceanic and Atmpospheric Administration, the National Weather Service provides daily weather reports for cities across the county. This is done through the use of 122 different Weather Forcast Offices throughout the country. These WFOs are responsible for the daily weather reports for serveral cities throughout their region of coverage. This data set takes the information from these WFO reports for cities across the country and summarizes it at the weekly level for all of 2016.

http://w2.weather.gov/climate/

Field Descriptions

Key List of... Comment Example Value
Average Integer number The average recorded temperature on this week, in degrees Farenheit. 39
City String The city that the reporting station sends its data to. Note that the recording station itself might actually be in a different city. "Birmingham"
Code String The unique code representing this station. "BHM"
Direction Integer number The average wind direction for that week, in degrees. 33
Full String A full string representation of the date this report was made. "2016-01-03"
Location String The exact location of this recording station. Note that the recording station itself might be in a different location than where it sends its data. "Birmingham, AL"
Maximum Integer number The highest recorded temperature on this week, in degrees Farenheit. 46
Minimum Integer number The lowest recorded temperature on this week, in degrees Farenheit. 32
Month Integer number The month of the year that this report was made. 1
Precipitation Real number The average amount of rain, in inches. 0.0
Speed Real number The average windspeed for that week, in Miles per Hour. 4.33
State String The state that the reporting station sends its data to. Note that the recording station itself might actually be in a different state. "Alabama"
Week of Integer number The first day of the week that this report was made. 3
Year Integer number The year that this report was made. 2016
